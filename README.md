# AI進化レーダー（毎日自動更新版）

使っているAIツールの最新ニュースを **毎日自動で集めて表示** するダッシュボードです。

## 仕組み（むずかしくないので安心してください）

1. GitHub が **毎日朝6時（日本時間）** に `update.js` を自動実行
2. `update.js` が各AIツールのニュース（GoogleニュースRSS等）を集めて `data.json` を更新
3. Netlify が変更を検知して **サイトを自動で再公開**
4. アプリ（`index.html`）は開くたびに `data.json` を読むので、いつ開いても最新

→ 一度セットアップすれば、あとは **何も操作しなくても毎日勝手に更新** されます。

## ファイル

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（これがサイトとして表示される） |
| `data.json` | 集めた最新ニュース（GitHubが毎日自動更新） |
| `update.js` | ニュースを集めて data.json を作るスクリプト（GitHubが実行） |
| `.github/workflows/update.yml` | 「毎日朝6時に実行」の予約設定 |

## セットアップ手順

詳しい手順は、チャットでClaudeが案内します。大まかには：

1. GitHub（無料）にこのフォルダ一式をアップロードして公開リポジトリにする
2. Netlify を「Import from Git」でそのリポジトリに連携する
3. GitHub の **Actions** タブで `Update AI feeds` を一度手動実行（Run workflow）
4. 完了。以後は毎日自動で更新されます

## 取得対象を増やす・変える

`update.js` の先頭の `TOOLS` リストに、ツール名とフィードURLを足すだけです。
迷ったら Claude に「このツールを追加して」と頼んでください。
